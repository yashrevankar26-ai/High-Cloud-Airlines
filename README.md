# High Cloud Airlines Data Analysis

## Project Overview
Analyzed operational and customer data of **High Cloud Airlines** to uncover insights that improve flight performance, revenue, and passenger satisfaction. Interactive dashboards provide actionable insights for business decision-making.

## Objectives
- Analyze flight delays, cancellations, and route performance
- Study passenger behavior and peak travel periods
- Identify revenue trends and key performance indicators (KPIs)
- Create interactive dashboards for operational and business insights

## Tools & Technologies
- **Excel**: Data cleaning, pivot tables, charts
- **SQL**: Querying, aggregations, joins, and KPI calculations
- **Power BI**: Dashboard creation, data modeling, KPI visualization
- **Tableau**: Advanced visualizations, trend and geographic analysis

## Key Insights
- Routes with highest delays and cancellations identified
- Peak travel periods and passenger demand analyzed
- Revenue trends across routes tracked
- Factors affecting on-time performance highlighted

## Outcomes
- Dashboards to support data-driven operational and business decisions
- Enhanced understanding of airline efficiency and passenger trends
- Recommendations to reduce delays and optimize routes
